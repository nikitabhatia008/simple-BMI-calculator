Simple BMI Calculator - Project Statement

Problem Statement
many people are unaware if their weight is healthy.BMI helps estimate weight category based on height and weight. This project creates a simple tool to calculate BMI and show health status.

Scope of the Project
*quick BMI calculation
*basic health intrpretation
*beginner-friendly python project

Target Users
*python beginners and students
*anyone interested in tracking their health
*educators needing a simpe python example

High Level Features
*input weight(kg) and height(m)
*calculate the BMI: BMI = weight/(height*height).
*show BMI rounded to 2 decimal.
*display health status: underweight,normal,overweight,obese.
