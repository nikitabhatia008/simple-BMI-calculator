# Simple-BMI-Calculator
A beginner-friendly python project that calculates Body Mass Index(BMI) based on user input and gives a basic health interpretation.

Features 
Calculates BMI from weight(kg) and height(m)
provides health status based on BMI value :
      * underweight
      * normal weight 
      * overweight
      * obese 
Simple,easy-to-understand python code
Beginner-friendly project suitable for learning python basics

Technologies used:
python
no external libraries required

Steps to install and run the project
1. **Clone the repository**
2. navigate to the project folder
3. run the python file
4. enter your weight and height when prompted,and view your BMI and health status.


Instructions for testing
1. run the program multiple times with different weight and height inputs.
2. check if the BMI calculation is correct using the formula:
   BMI = weight/(height*height)
3. verify that the health status output matches the BMI range:
   *underweight : BMI < 18.5
   *normal : 18.5 <=BMI < 24.9
   *overweight : 25 <= BMI < 29.9
   * obese : BMI >=30
